Морзянка на лету

Python-утилита для Windows 11, которая автоматически перехватывает ввод русской клавиатуры и мгновенно печатает соответствующий код Морзе. Без копирования. Без лишних действий.

Python Скрипт

morse_utility.py 
import keyboard
import time

# Словарь соответствия русских букв азбуке Морзе
MORSE_DICT = {
    'А': '.-', 'Б': '-...', 'В': '.--', 'Г': '--.', 'Д': '-..',
    'Е': '.', 'Ё': '.', 'Ж': '...-', 'З': '--..', 'И': '..',
    'Й': '.---', 'К': '-.-', 'Л': '.-..', 'М': '--', 'Н': '-.',
    'О': '---', 'П': '.--.', 'Р': '.-.', 'С': '...', 'Т': '-',
    'У': '..-', 'Ф': '..-.', 'Х': '....', 'Ц': '-.-.', 'Ч': '---.',
    'Ш': '----', 'Щ': '--.-', 'Ъ': '--.--', 'Ы': '-.--', 'Ь': '-..-',
    'Э': '..-..', 'Ю': '..--', 'Я': '.-.-',
    '1': '.----', '2': '..---', '3': '...--', '4': '....-', '5': '.....',
    '6': '-....', '7': '--...', '8': '---..', '9': '----.', '0': '-----',
    ' ': '/'
}

enabled = True

def on_press(event):
    global enabled
    
    # Переключение работы по нажатию Alt+M
    if event.name == 'm' and keyboard.is_pressed('alt'):
        enabled = not enabled
        state = "ВКЛЮЧЕН" if enabled else "ВЫКЛЮЧЕН"
        print(f"Статус: {state}")
        return

    if not enabled:
        return

    # Проверяем, является ли символ русской буквой или цифрой
    char = event.name.upper()
    if char in MORSE_DICT:
        # Стираем введенный символ (backspace)
        keyboard.press_and_release('backspace')
        # Печатаем код Морзе
        keyboard.write(MORSE_DICT[char] + ' ')

print("Утилита win11-morse-code-russian запущена!")
print("Используйте ALT + M для включения/выключения")
print("Нажмите ESC для выхода")

keyboard.on_press(on_press)
keyboard.wait('esc')

Инструкция по публикации на GitHub

1. Подготовка окружения

  • 1 Установите Python с сайта python.org
  • 2 Откройте терминал (CMD или PowerShell)
  • 3 Установите библиотеку: pip install keyboard

2. Публикация в репозиторий

Структура вашего проекта:

  • main.py (основной код)
  • requirements.txt (текст: keyboard)
  • README.md (описание)

Команды для терминала:

git init
git add .
git commit -m "initial release"
git branch -M main
git remote add origin https://github.com/eflecto/win11-morse-code-russian.git
git push -u origin main
